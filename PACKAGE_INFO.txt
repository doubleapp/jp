================================================================================
QUICK INSTALLATION
================================================================================

1. Run: install.bat
2. Close and reopen CMD
3. Run: jp add myproject C:\path\to\project
4. Run: jp myproject

================================================================================
FEATURES
================================================================================

✓ Add directory shortcuts with custom names
✓ Jump to directories instantly across different drives
✓ Automatic cross-drive detection (C: → E:, etc.)
✓ Visual feedback when switching between drives
✓ List all saved shortcuts
✓ Remove shortcuts when no longer needed
✓ Persistent storage across sessions
✓ Error handling for inaccessible directories
✓ TAB completion in CMD (via Clink) and PowerShell (built-in)
✓ No dependencies or admin rights required
✓ Works on Windows 7 and later
✓ Supports paths with spaces
✓ Case-insensitive shortcut names

================================================================================
SYSTEM REQUIREMENTS
================================================================================

- Operating System: Windows 7 or later
- Shell: Command Prompt (cmd.exe)
- Disk Space: < 100 KB
- Permissions: User-level (no admin required)
- Dependencies: None

INSTALLATION LOCATIONS
================================================================================

After installation, files will be located at:

- Program: %USERPROFILE%\bin\jp.bat
